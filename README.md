## Project Overview
This project involves cleaning and processing a raw Amazon Product dataset using Python and the Pandas library. The primary objective is to transform unstructured, messy e-commerce data into a standardized format ready for exploratory data analysis and dashboard visualization.

## Key Activities
- **Data Loading:** Imported raw CSV data into a Pandas DataFrame.
- **Handling Null Values:** Identified and managed missing data points (e.g., missing product ratings or prices) to ensure statistical accuracy.
- **Removing Duplicates:** Cleared redundant product listings to prevent skewed analysis.
- **Formatting:** Standardized text columns and ensured correct data types for pricing and dates.

## Technologies Used
- **Python:** The primary programming language.
- **Pandas:** Used for data manipulation, filtering, and cleaning.
- **Google Colab:** Cloud-based Jupyter Notebook environment used for execution.
